# Phase C 标注质检报告模板（C-06）

> 版本 V1.0 | 2026-07-10 | 任务 C-06（泳道 B｜数据工程）
>
> 规范源：`docs/PHASE_C_ANNOTATION_GUIDE.md`（判定准则、度量定义、验收门槛）
> 用法：**每批金标签复制本模板一份填写**，与金标签一同回流 C-07；**无质检报告的批次不得入库**。
