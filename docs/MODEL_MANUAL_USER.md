## 1. 这是什么:一分钟看懂

「工程模型」把一个项目里已上传的图纸,**自动升级成一个可在浏览器里旋转查看的三维模型**:楼层按标高一层层堆叠,结构柱、墙、梁、板、机电管线和设备按图纸真实位置重建出来,AI 审图发现的问题、跨图冲突也会以彩色标记球挂在对应楼层上。

你可以用它来:

- **整体感知项目**:一眼看清有几栋楼、多少层、哪层有多少问题。
- **定位问题**:点选三维里的红色标记球,直接跳到出问题的图纸。
- **核对构件**:查看柱/墙/梁/板的位置、数量、截面。
- **参与审校**:对机器识别不确定的地方做「确认/否定/改类/补框」,你的每一次判断都会让模型越来越准。
- **看工程量**:查看混凝土、模板、钢筋的算量汇总。

> **重要认知**:这是「AI 出初模 + 人工在平台内审改」的工作方式,**不是「一键出成品 BIM」**。机器负责把 80% 的确定性活儿干完,不确定的部分交给你快速核对。整体效率提升现实区间约 **25%–30%**。

## 2. 术语速览

| 术语 | 通俗解释 |
|---|---|
| **构件** | 柱、墙、梁、板、管线、设备这些建筑组成部分 |
| **单体** | 一个项目里的一栋楼(如 1# 楼、裙房) |
| **楼层归属** | 一张图纸属于哪栋楼的哪一层 |
| **标记球** | 三维里悬浮的彩色小球,代表一个审图问题或跨图发现,颜色越红越严重 |
| **LOD** | 模型精细度等级。审图骨架(粗)→ 建筑体量(中)→ 实景近似(细) |
| **渲染模式** | 三维看的方式:IFC 模型 / 构件 / 贴图 / 混合 |
| **符号标注** | 对图纸上的门、窗、柱等图元画框、定类别 |
| **置信度** | 机器对自己识别结果有多大把握,0~1,越低越需要你把关 |
| **返工点** | 你在审校里做了「改类/否定/补框」= 机器初模需要人工修正的地方 |

## 3. 快速上手(5 分钟跑通)

1. 左侧主菜单点击 **「工程模型」**。
2. 在项目卡片网格里点你要看的项目。
3. 若提示「该项目尚未生成工程模型」,点 **「立即生成模型」**,页面进入「构建中」并每 5 秒自动刷新进度条。等待其变为「已就绪」(取决于图纸数量,通常数分钟)。
4. 模型就绪后,中间出现三维视图:**左键拖拽旋转,滚轮缩放,右键平移**。
5. 点一个红色标记球 → 右侧抽屉弹出问题详情 → 点「查看图纸」直达源图。
6. 右栏若显示「待人工识别 N」,展开处理一两条,体验「确认/改类」。

完成 ✅ 你已经会用核心功能了。下面是逐项详解。

## 4. 四个入口:从哪里打开工程模型

工程模型对**所有已登录用户开放**,不限角色。共有四个入口,都会打开同一个模型工作台:

| 入口 | 位置 | 说明 |
|---|---|---|
| **主菜单** | 左侧菜单「工程模型」 | 先进项目选择器,再点项目卡片 |
| **图纸详情页** | 图纸详情右上「在工程模型中查看」 | 打开模型并**聚焦到当前这张图纸**(相机对准 + 高亮描边) |
| **项目看板** | 项目看板顶部「工程模型」按钮 | 未选项目时按钮禁用 |
| **套图审查详情** | 跨图发现卡片右上「在模型中查看跨图发现」 | 仅当该批次审查完成后出现 |
| **图纸列表** | 列表页「工程模型」按钮 | 读取筛选栏已选项目;未选会提示「请先选择项目」 |

## 5. 主界面导览

模型工作台分为**顶部状态条**和**三栏主体**。

### 5.1 顶部状态条

- **状态徽标**:构建中(蓝色转圈)/ 已就绪(绿色)/ 构建失败(红色)。
- **版本与时间**:`版本 v3`、`构建于 …`。
- **「重建模型」按钮**:重新跑一遍构建(构建中时禁用)。见 [§6](#6-生成与重建模型)。
- **统计标签**:图纸数、问题数、楼层数、`待人工识别 N`、重建模式(构件级/贴图级/混合)、构件总数、各严重度问题数量。
- **LOD 模式按钮组**:审图骨架 / 建筑体量 / 实景近似。不可用的等级会置灰,鼠标悬停有原因说明。
- **渲染模式切换**(当有多种可选时出现):IFC 模型 / 构件 / 贴图 / 混合。
- **构建中**会显示进度条;**失败**会显示红色错误提示。

### 5.2 三栏布局

| 栏 | 宽度 | 内容 |
|---|---|---|
| **左栏** | 窄 | 语义树、单体、楼层、专业、严重度、标记类型、构件图层等过滤器 |
| **中栏** | 自适应 | 三维视图(主舞台) |
| **右栏** | 宽 | 模型质量、语义审查、待人工识别三大工作台 |

## 6. 生成与重建模型

### 首次生成

项目从未建过模型时,中间显示空态「该项目尚未生成工程模型」,点 **「立即生成模型」**。

### 重建

图纸有更新、做了人工标注、或想应用最新算法时,点顶部 **「重建模型」**。

- 触发后状态变「构建中」,进度条每 5 秒自动刷新,阶段依次为:抓取输入 → 渲染图纸 → 识别构件 → 组装模型。
- 完成后版本号 +1(如 v3 → v4),旧版本自动被替换。
- **构建是后台异步进行的,你可以离开页面**,稍后回来查看结果。

> 💡 你做的每一次楼层标注、符号确认,都会在下次重建时被吸收进模型。**「标注 → 重建 → 更准」是推荐的迭代节奏。**

## 7. 三维视图操作

### 7.1 基本视角操作(所有渲染模式通用)

| 操作 | 方式 |
|---|---|
| 旋转 | 左键拖拽 |
| 缩放 | 鼠标滚轮 |
| 平移 | 右键拖拽 |
| 选中构件/标记 | 单击(系统会区分单击和拖拽,拖拽超过约 6 像素不算点选) |

视角带阻尼效果,松手后会平滑停下。

### 7.2 四种渲染模式

顶部「渲染模式」切换(具体可选项取决于该模型生成了哪些数据):

- **IFC 模型**:高性能 Fragments 渲染,面向大体量模型,可点选构件查看 IFC 属性。
- **构件**:显示重建出的柱/墙/梁/板等挤出实体。
- **贴图**:把每张图纸作为面板贴在对应楼层。
- **混合**:构件 + 贴图叠加显示。

切换渲染模式时,**相机视角会保留**,不会跳回原点。

### 7.3 LOD 精细度

顶部「LOD 模式」按钮:**审图骨架**(最粗,看结构关系)、**建筑体量**(中等)、**实景近似**(最细)。等级越高需要模型数据越完整,不满足条件的等级会置灰并给出原因。

### 7.4 楼层 / 单体隔离

在左栏点击某个**楼层**或**单体**即可「隔离」——只高亮它,其余变半透明;**再点一次取消**隔离。方便你聚焦看某一层或某栋楼。

### 7.5 IFC 属性面板(IFC 模型模式下)

点选构件后,右上角浮出属性面板:IFC 类型、GUID、名称,以及分组的 Pset 参数。点 **「语义树定位」** 可在左栏语义树中反查该构件位置。

## 8. 左栏:过滤与隔离

左栏用于收窄你在三维里看到的内容,**都是即时生效的显示开关,不改变模型数据**:

- **语义树**:按单体/分区/功能空间层级浏览与定位。
- **单体列表**:点击隔离某栋楼;标签标注「人工/识别/无几何」。
- **楼层列表**:按楼层高低倒序排列,点击隔离。
- **专业**:勾选结构/建筑/机电/装修,过滤构件与标记。
- **严重度**:勾选严重/较大/一般/提示,过滤问题标记球。
- **标记类型**:图内问题 / 跨图发现。
- **构件图层**(构件模式下):柱/墙/梁/板/管线/设备/外观壳体的显隐勾选。

## 9. 点选与详情抽屉

在三维里单击对象,右侧弹出详情抽屉,分三种:

- **图纸信息**:图号、图名、专业、阶段、问题数 → 按钮「进入图纸详情」跳转 `/drawings/{id}`。
- **问题标记**:严重度、类型、楼层、专业代码、所属图纸 → 按钮「查看图纸」。
- **构件信息**:构件类型、楼层、数量、标注,并提示「构件级重建(矢量图纸提取)」→ 按钮「查看来源图纸」。

## 10. 右栏工作台一:待人工识别

当机器对「这张图属于哪层」或「这个符号是什么」不确定时,会进队列请你把关。顶部用分段控件切换两个 Tab。

### 10.1 Tab「楼层归属」

对无法自动判断楼层的图纸补标签:

1. 每条显示缩略图/文本线索、图号、置信度。
2. 填三个下拉(支持自动补全):**单体**(可新建/重命名)、**楼层**、**图纸类型**(平面图/立面图/剖面图/节点详图/机电平面)。
3. 三项都填好后点 **「保存标注」**,该条从队列移除。

### 10.2 Tab「符号标注」

对图纸上的门窗柱等图元逐个把关:

1. 上方下拉选一张图纸,图上会叠加彩色候选框。
2. **框颜色 = 置信度**:🔴 红 < 0.5(最需审)、🟡 黄 < 0.75、🟢 绿(较可靠)。系统按低置信优先排序。
3. 对每个候选可以:
   - **确认**:识别正确。
   - **否定**:识别错误。
   - **改类**:类别选错了,下拉改成正确类别(共 9 类:柱/梁/板/墙/门/窗/管道/设备/轴线)。
4. 若机器漏识别,点 **「补框」**,选类别 + 填框的四个坐标 → 「新增框」。

> ✅ 你的「确认 / 改类 / 补框」结果会成为**金标签**,用于持续训练识别模型——**审校即产数据**。

## 11. 右栏工作台二:语义审查

面向模型的语义正确性(拓扑闭合、构件命名、规范符合),分两区。

### 11.1 成果审校(拓扑/命名/规范)

- 队列每条标注类别(拓扑闭合/构件命名/规范符合性/构件/符号)、置信度,冲突项左侧有红色边条,并标「规则-模型冲突/未闭合」。**低置信 + 冲突的会排在最前**。
- 三个动作按钮:**确认** ✓ / **改类** ⇄(弹窗选新类别)/ **否定** ✗。

### 11.2 语义候选队列(单体/分区/功能空间)

- 每条显示类型(单体/分区/功能空间/施工分区)、状态、置信度。
- 七个操作:**查看证据**、**确认**、**调整父级**、**重命名**、**合并**、**拆分**、**拒绝**。
- 做「重命名/合并/调整父级/拆分」时,弹窗会**实时预览影响范围**(会影响哪些节点/图纸/楼层),确认无误再提交。
- 若期间别人也改了语义树,提交会提示「语义树版本已更新」,点「刷新到最新版」后重做即可。

## 12. 右栏工作台三:模型质量与返工度量

### 12.1 模型质量

六个指标一览:未分层、楼层冲突、低置信度单体、待人工识别、待审语义、语义冲突。楼层冲突会用提示条列出前几条。

选中某个构件/范围时,还会显示 **LOD 质量**:当前 LOD 等级、已通过的门槛(绿标)、缺失的证据(橙标)、以及为什么降级的说明。

### 12.2 返工收敛度量

- 可按专业筛选(结构/机电/装修/建筑)。
- 四个比率卡片:**确认率**(绿,越高越好)/ **改类率**(黄)/ **否定率**(红)/ **补框率**(蓝)。
- 各专业**返工率进度条**(≥50% 红、≥25% 黄、其余绿)、各类别明细表、按天的返工率收敛趋势图。
- **口径**:返工点 = 改类 + 否定 + 补框;确认 = 机器初模一次通过。趋势下降说明模型在变准。这里的数字对应「25%–30% 效率提升」的现实区间。

## 13. 查看工程量(算量结果)

模型能对混凝土、模板、钢筋做算量汇总,按项目 / 楼层 / 单体三级呈现:

- **混凝土体积**:已扣减梁柱板相交部分的**净体积**(比毛体积小)。
- **模板量**:接触面积。
- **钢筋量**:需有配筋输入才计算;没有配筋数据时会明确标注「缺配筋」而不会瞎猜。

若算出**钢筋优化有正节约**,可一键生成一条**创效提案草稿**,进入正常的三审创效流程(经济师签字等硬约束照常执行,不被绕过)。

> ⚠️ 算量为工程辅助参考。变截面、地区细则等特殊情况按 MVP 通用口径处理,正式结算仍需人工复核。

## 14. 常见问题 FAQ

**Q:点了「立即生成模型」很久还在转圈?**
A:构建是后台异步的,时间取决于图纸数量与类型。可离开页面稍后再来;进度条每 5 秒刷新。若长时间卡住或显示「构建失败」,联系管理员查看后台日志。

**Q:三维里有些图纸是白框/线框,没有贴图?**
A:该图渲染失败或过大时会用线框占位,属正常降级,不影响其余部分。

**Q:为什么某层是空的、构件很少?**
A:可能该层图纸是扫描件、非矢量,或未正确归属楼层。可在「待人工识别 → 楼层归属」补标注后重建。

**Q:改了标注,模型没变?**
A:标注保存后需点顶部「重建模型」才会应用到三维。

**Q:构件位置/截面不准?**
A:结构柱/墙/板较可靠;梁为近似(截面可能取默认值);机电为体块示意。有剖面图时系统会恢复真实层高与截面,精度更高。请以图纸为准,在审校里修正。

**Q:我没有管理员权限,能用这些功能吗?**
A:能。工程模型的查看、生成、标注、审校对所有登录用户开放,不需要特殊角色。
