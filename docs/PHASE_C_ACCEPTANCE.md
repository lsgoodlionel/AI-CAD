## 0. 能力边界（诚实先行，向上管理预期）

**全自动生产级 BIM 今天不存在。** Phase C 的定位是「**AI 出初模 + 平台内人工审改**」，
**非「一键出 BIM」**。全流程效率提升现实值 **25–30%**，任何 KPI/对外话术均以此为锚，
不承诺自动出图。学术峰值实例精度 ~94%，但 BIM 要求拓扑闭合、语义正确、可结算，
单点错误在整栋楼累积成大量返工——故**符号识别学习模型只补规则的模糊边界，不替代规则**，
人工审校工作台把返工点收敛。

## 4. 遗留风险与顺延项（诚实清单）

1. **M1 终评卡 C-09**：真实符号识别精度需 GPU + C-07 脱敏数据 + CADTransformer 权重微调；
   当前 model 端 mock 占位。**这是关键路径的最后一公里**，基座已备，只待数据/算力注入。
2. **C-11 SymPoint 天花板**：需 GPU + `research/sympoint-eval/` 隔离环境，仅回流数字。
3. **数据冷启动成本**：中文专业域自建数据集（结构→机电→装修）是最大隐形成本，滚动持续投入。
4. **签字栏测试预设**：默认密码 000000 为测试用，上线前须重设。
5. **前端视觉验证顺延**：审校工作台 UI 已实现并类型对齐，浏览器视觉/交互验证需真实环境补跑。
6. **线型符号匹配**：评测中线型符号（梁/管）零面积 bbox 影响 IoU 匹配，真实框状符号金标签不受影响，
   但线型构件识别口径需在 C-09 数据标注时明确（框标注约定）。
