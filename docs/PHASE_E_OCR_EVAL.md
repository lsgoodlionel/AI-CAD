## 结论

## 标高质量深核

- **合理范围**:4808/4919(**97.8%**)标高 ∈ [-50, 100]m;2.2% 离群(把尺寸/范围文本
  误读,如 970m/-710m)。section-z 下游已按置信门槛 + 主序 RANSAC 过滤,离群不入模型。
- **真值抽样**(围护体剖面图(一)):16.200 / 9.300 / 5.500 / -0.840 / -1.050 / -2.050
  **全部正确**,置信 0.97~1.00。
- **已知小瑕疵**:「标高范围 -1.000~-24.800」这类区间文本被解析为单值(-1.000);
  量小,不影响主序标定。

## 调优建议(增量,未强制实施)

1. **标高范围文本**:classify 增「A~B」区间识别,拆成端点或标记 range(减少单值误读)。
2. **轴号去噪**:短单字母(M/E/S)与真轴号难分;可结合几何轴线位置交叉校验
   (与档案坐标变换 A1 共用),提升 axis 高置信占比。
3. **离群标高**:数值 > 100m / < -50m 直接降为低置信(section-z 已过滤,此为提前标注)。

## 边界

- 本评测基于 OCR **自身置信 + 值合理性 + 抽样真值**,非全量人工 IoU/Kappa 金标签
  (那是独立标注数据工程,见 `docs/PHASE_C_ANNOTATION_GUIDE.md` 范式)。
- 回填完成(2309 图全覆盖)后数字会更全,但当前 300 图样本已足够代表性。
